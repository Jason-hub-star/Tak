# Session Retro (탁디장)

## Capture

- ✓ 성공 패턴: 무엇이 빨랐나
- ✗ 삽질 회피: 뭘 늦게 발견했나
- 🔄 반복 수작업: 자동화 기회는
- 📝 사용자 교정: 뭘 놓쳤나

## Output

- `ai-context/worklog.md` — 상세 일지 (후속 검토용)
- `docs/status/PROJECT-STATUS.md` — 1줄 진도 업데이트
- `DECISION-LOG.md` — 구조 결정 있으면 기록
- 필요 시: 새 하네스 또는 자동화 아이디어

## Example

**성공**: EmailJS 타입 체크가 빨리 컴파일 오류를 잡았다.  
**교훈**: 번들 크기 체크 자동화 추가 고려.  
**다음**: media-performance-budget 하네스 준비.
