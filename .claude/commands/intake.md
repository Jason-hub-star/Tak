# /intake — 작업 시작

새 세션이나 큰 작업 시작 시 필요한 컨텍스트를 한번에 로드한다.

## 순서

1. `AGENTS.md` — 로딩 순서와 규칙 파악
2. `docs/prd.md` — 제품 기준 (최우선)
3. `docs/status/PROJECT-STATUS.md` — 현재 상태
4. `ai-context/master-plan.md` — 우선순위 백로그
5. 작업 관련 하네스 (harnesses/REGISTRY.md 참고)

## 체크리스트

- [ ] 문서 로딩 완료
- [ ] 검증 명령 이해 (`npm run build`)
- [ ] 다음 액션 확인 (PROJECT-STATUS에서)
- [ ] 필요 하네스 결정

## 시간 예상

- 첫 진입: 10~15분 (읽기)
- 재진입: 3~5분 (PROJECT-STATUS + 필요 하네스만)
