① 필수

구글 서치콘솔 소유확인 코드

  1. 1search.google.com/search-console 접속 → 구글 로그인
  2. 2속성 추가에서 “URL 접두어” 선택 → https://takdijang.com 입력
  3. 3소유확인 방법 중 “HTML 태그” 펼치기
  4. 4<meta name="google-site-verification" content="이부분" /> 에서 content 안의 값만 복사 (따옴표·태그 빼고)
→ 이렇게 넘겨주세요 (값만 바꿔서):
GOOGLE_SITE_VERIFICATION=여기에_복사한_content_값

② 필수 · 한국 핵심

네이버 서치어드바이저 소유확인 코드

  1. 1searchadvisor.naver.com 접속 → 네이버 로그인 → 웹마스터 도구
  2. 2사이트 등록란에 https://takdijang.com 입력 후 추가
  3. 3소유확인 방법에서 “HTML 태그” 선택
  4. 4<meta name="naver-site-verification" content="이부분" /> 의 content 값만 복사
→ 이렇게 넘겨주세요:
NAVER_SITE_VERIFICATION=여기에_복사한_content_값
💡 소유확인을 누르기 전에 제가 코드를 배포해야 태그가 보여요. 값 주시면 바로 반영하고, 그다음 “확인” 누르시면 됩니다.

④ 선택 · 색인 가속

IndexNow 키 — 나중에 해도 OK

새 글·수정 페이지를 네이버/Bing에 즉시 통보하는 기능이에요. 안 해도 검색은 됩니다.

  1. 1아무 영문/숫자 32자 이상 키를 하나 정해요 (비밀번호 만들듯)
  2. 2그 키를 제게 주시면 public/<키>.txt 파일로 올릴게요
  3. 3이후 npm run seo:indexnow 한 번으로 전체 URL 통보 완료
→ 이렇게 넘겨주세요:
INDEXNOW_KEY=내가_정한_32자_이상_임의키
